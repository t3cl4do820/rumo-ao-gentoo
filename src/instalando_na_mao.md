# Instalando ele na mao

Agora sim, instalando ele sem script, confesso que esta sendo um pouco desafiador pois nao conheco alguns conceitos sobre linux e afins, entao quando eu vejo algo novo e paro a instalacao e vou estudando...

Estou usando [qemu](https://www.qemu.org/) para emular o void antes de instalar na minha maquina real.

Recomendo ler a doc oficial do void: https://docs.voidlinux.org/

E mais uma coisa, a doc do void linux presume que voce nao seja igual a mim (um leigo kk), logo nao vai ter tudo mastigado para voce entender o que esta acontecendo.


### Ja dento da ISO

Eu recomendo voce bootar a iso `base` (que eh a sem interface grafica), no inicio do modo live voce precisa logar:

`login: root`
`password: voidlinux`

Agora sim voce esta dentro do modo live para dar comando, voce vai estar usando o shell padrao [sh (Bourne Shell)](https://pt.wikipedia.org/wiki/Bourne_shell) que foi o shell padrao no Unix alguns anos atras.

Se voce quiser pode rodar `bash` para entrar no bash e prosseguir com a instalacao.

Agora sim pode ir pra proxima pagina.
